# Installation

## Development

## Production

1. Install latest Node and NPM

### Frontend

1. [Install Angular CLI](https://angular.io/cli)

```sh
npm install -g @angular/cli
```

2. Start project

```sh
ng new frontend
```

### Backend

1. [Install NestJS CLI](https://docs.nestjs.com)

```sh
npm i -g @nestjs/cli
```

2. Start project

```sh
nest new backend
```
